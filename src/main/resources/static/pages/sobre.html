Sobre

A API MyMovieList foi desenvolvida como parte de um projeto acadêmico na Fatec Mogi Mirim, com foco em oferecer uma solução prática e eficiente para consulta de filmes, diretores e sinopses. Construída com Spring Boot, a API aplica boas práticas de desenvolvimento e segurança, como autenticação JWT e versionamento de código, além de contar com suporte a contêineres e hospedagem em nuvem. Nosso objetivo é combinar tecnologia moderna com simplicidade para entregar uma experiência funcional e bem estruturada.
